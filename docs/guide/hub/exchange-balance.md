# 中转站余额兑换教程






### 购买

在[卡网](https://pay.ldxp.cn/shop/qingwaAA) 选择合适的想要充值的余额，进行购买。购买后会自动发放**兑换码**。

![image-20260714223018276](../img/image.png)



### 

### 充值/兑换

#### 注册/登录

浏览器(推荐谷歌浏览器)打开[中转地址](https://hub.ccttt99.com/) 进行注册登录



#### 兑换

进入中转后台后，点击左侧菜单 `兑换` ，将在`卡网`购买的兑换码输入在输入框中，并点击`提交`即可
