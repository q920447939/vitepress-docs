# GPT  封号后的申诉措施

所有申诉都使用英文！

1. GPT会发一封封号邮件，先在邮件里面发起申诉(一般是直接驳回)；
2. 到`helper center `右下角 聊天窗口对话触发工单 ，要发很多次 可能才会触发邮件回复问题解决了没有，如果触发了工单回复，那么提交信息；
3. GPT[论坛](https://community.openai.com/) 中发起帖子申诉；
4. 给官方发邮件 （`support@openai.com`）



注:


        1.先在邮件里面查看自己的行为有没有被官方识别为 cyber等

        2.在申诉时，尽可能的把资料填写详细，比如你的对话记录等

        3.封号是官方的行为，有可能是误封，当然也有可能你做了反代，逆向，破限违规操作



## 退款
如GPT官方同意退款，那么会退到虚拟卡上(具体到虚拟卡上的时间不一定，一般为2周)，
在虚拟卡上收到退款后，收取18%技术服务费,剩余金额将进行退还。
